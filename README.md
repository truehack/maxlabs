# Лабораторная работа АиСД №1


## Запуск в powershell

1. git clone https://github.com/truehack/maxlabs.git
2. cd maxlabs
3. gcc binarytree.c -o binarytree.exe
4. .\binarytree.exe

## Инструкция по пользованию
1. Add node          - добавить узел в дерево
2. Delete node       - удалить узел из дерева
3. Show tree         - вывести дерево на экран
4. Execute task      - вычислить значение функции
5. Clear tree        - очистить дерево
0. Exit              - завершить работу программы
